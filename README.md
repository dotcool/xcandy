# 糖果派消消乐-休闲消除游戏

《糖果派消消乐》是一款经典的休闲游戏，我们会看到高清的游戏画面、美妙的音效，《糖果派消消乐》玩法：单指操控，三个及以上同色糖果方块可以消除；无时间限制，达到要求即可过关。未达到分数可利用道具帮助通关。
